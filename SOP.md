PURPOSE
Operate a calm, bounded intake system for surplus items (RIA instruments, auto
parts, vintage electronics). Filter noise. Preserve time. Prioritize seriousness.

DESK DEFINITION
The Parts Desk is an intake gate, not a storefront.
All inbound requests flow through the desk. No exceptions.

INBOUND CHANNELS
1) Web inquiry form
2) Desk voicemail (Google Voice)
No DMs. No SMS. No live calls.

SERVICE LEVEL
- Responses within 24 hours (one business day)
- Order handled: first complete, serious inquiry
- No urgency escalation

VOICEMAIL HANDLING
- Do not answer live
- Log timestamp, caller, item reference, callback number
- Ignore messages without clear item reference
- Transcription acceptable; verify by listening if unclear

FORM HANDLING
Required fields:
- Item reference
- Contact info
- Location / shipping preference
Optional but positive signal:
- Intended use

Incomplete forms may be ignored.

INITIAL RESPONSE TEMPLATE
Use neutral tone. No negotiation language.
Acknowledge receipt.
Confirm availability.
Outline next step (pricing, shipping, handoff).
Do not promise holds.

CALLBACK POLICY
- Outbound calls may originate from personal number
- This does not grant future direct access
- If caller attempts bypass later, redirect to desk politely

NEGOTIATION POLICY
- Prices are firm if listed
- No back-and-forth bargaining
- If price objection arises: thank them and close loop

HOLDS
- No holds without explicit confirmation
- Time-bound if granted (admin to confirm duration)

ESCALATION
Only escalate to owner if:
- High-value item
- Provenance-sensitive inquiry
- Exceptional alignment / seriousness

PLATFORMS
- Canonical listing lives at /parts
- External platforms may only link to desk
- Never list items directly on noisy platforms

LANGUAGE RULES
- No sales language
- No urgency cues
- No apologies
- Clear, declarative sentences only

PHILOSOPHY (DO NOT EXPLAIN)
This desk favors seriousness, patience, and mutual respect.
Objects move when timing is right.

END SOP
